Leetcode 2677. Chunk Array, slice 함수

Posted on 2024-02-06 In algorithm

맨날 알고리즘 문제를 파이썬 또는 자바로만 풀다가 처음 leetcode에서 자바스크립트를 통해서 풀게 되었다. 2677. Chunk Array, slice 함수 문제는 배열을 원하는 크기 만큼 분할하여 배열에 저장하는 문제인데, 처음에는 slice() 라는 함수의 존재 여부도 모르고 문제를 풀었다. 다른 사람들의 풀이를 보다가 slice() 라는 메서드의 존재 여부를 알게되었다. 따라서 이번 글에서는 이 문제의 풀이와 slice() 메서드의 작동방식을 비슷하게 구현해볼 것이다.

문제

문제는 여기(Leetcode 2677)에서 확인할 수 있다. 이 문제는 앞서 언급했듯이 배열을 원하는 크기 만큼 분할하여 배열에 저장하는 문제이다.

예를 들어 설명하자면 배열과 분할 할 때 원하는 크기를 다음과 같이 입력 받았다고 가정하자.(arr = [1,9,6,3,2], size = 3)

이때 output으로는 [[1,9,6],[3,2]] 가 반환이 된다. 이 예시처럼 size 대로 분할하고 모자르면 모자른 대로 그룹을 만들면 된다. 만약 size>=arr.length 라면 arr을 그대로 담은 배열(즉, 2차원 배열)을 반환하면 된다.
