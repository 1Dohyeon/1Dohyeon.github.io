민코프스키 거리(Minkowski Distance)

두 개의 p차원 데이터 객체 i=(xi1, xi2, ..., xip), j=(xj1, xj2, ..., xjp) 사이의 거리를 계산하는 일반적인 방법이다. 민코프스키 거리는 다음과 같이 정의된다.

여기서 h는 거리 측정의 차수를 나타내며, h의 값에 따라 다양한 형태의 거리를 얻을 수 있다.

민코프스키 거리의 특성

  • d(i,j) > 0 (단, i≠j), d(i,i)=0: 양의 정부호성
  • d(i,j) = d(j,i): 대칭성
  • d(i,j) ≤ d(i,k)+d(k,j): 삼각 부등식

이러한 특성을 만족하면 해당 거리는 메트릭(metric)이라 불린다.

민코프스키 거리의 특수한 경우들

  • h=1: 맨해튼 거리(Manhattan Distance) 혹은 L1 거리
    • 예를 들어, 해밍 거리(Hamming Distance)는 두 이진 벡터 간에 다른 비트의 개수를 의미한다.
  • h=2: 유클리드 거리(Euclidean Distance)
    • 가장 일반적으로 사용되는 거리 측정 방법이다.
  • h→∞: 최댓값 거리(Supremum Distance) 혹은 L∞ 거리
    • 이 경우, 벡터 간의 모든 성분 중 최대 차이 값을 거리로 취한다.

민코프스키 거리의 예시

데이터 포인트들은 각 속성의 값을 가지며, 다양한 h 값을 적용하여 거리 행렬을 계산한다. 예를 들어, 4개의 데이터 포인트 x1,x2,x3,x4가 다음과 같을 때:

Point Attribute 1 Attribute 2
x1 1 2
x2 3 5
x3 2 0
x4 4 5

계산 결과

  • L1 (맨해튼 거리): d(x1,x2)=|1−3|+|2−5|=5
    • 이 방식으로 다른 모든 쌍에 대해 거리를 계산한다.
  • L2 (유클리드 거리): d(x1,x2)=√((1−3)^2+(2−5)^2)≈3.61
  • L∞ (최댓값 거리): d(x1,x2)=max(|1−3|,|2−5|)=3

이렇게 각각의 거리 계산 방법에 따라 데이터 간의 거리를 다르게 측정할 수 있다. 이러한 민코프스키 거리는 다양한 상황에서 데이터 간의 유사성이나 비유사성을 측정하는 데 사용된다.
