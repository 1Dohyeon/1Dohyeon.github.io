Filter Methods에서의 특징 선택 과정

set of all features -> selecting the best subset -> learning algorithm -> performance

Set of all features:

  • 이 단계에서는 데이터셋에 있는 모든 특징(변수)이 준비된 상태다. 즉, 원래 데이터에 포함된 전체 특징들이 포함된다.

Selecting the best subset:

  • 이 단계는 특징 선택(Feature Selection) 과정이다. 여기서 모든 특징 중에서 가장 중요한 특징들만 골라낸다. 필터 방법, 래퍼 방법, 임베디드 방법 중 하나를 사용해 가장 예측에 도움이 되는 특징들의 하위 집합을 선택하는 것이다.

Learning algorithm:

  • 선택된 특징들로 머신러닝 알고리즘을 훈련하는 단계다. 이 알고리즘은 주어진 데이터를 학습해, 목표 변수(예측하고자 하는 값)를 예측하는 모델을 만든다.

Performance:

  • 마지막으로, 학습된 모델의 성능을 평가한다. 성능 평가는 일반적으로 정확도, 정밀도, 재현율 등의 지표를 사용해 이루어진다. 여기서 선택된 특징들이 모델 성능에 얼마나 긍정적인 영향을 미쳤는지 확인할 수 있다.
