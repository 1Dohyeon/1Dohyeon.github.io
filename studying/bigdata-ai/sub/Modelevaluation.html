Model evaluation(모델 평가)

종류

  • 훈련 세트(Training set): 모델 파라미터 추정.
  • 검증 세트(Validation set): 모델 선택.
  • 테스트 세트(Test set): 최종 평가.

각 데이터 세트는 머신러닝 모델을 학습하고, 검증하고, 최종 성능을 평가하는 데 사용된다.

기본적인 훈련-테스트 구조

기본적인 훈련-테스트 구조

Training set ← Model → Test set:

  • 이 부분은 가장 기본적인 모델 학습 및 평가 방법이다. 훈련 세트(Training set)에서 모델을 학습한 후, 테스트 세트(Test set)에서 모델의 성능을 평가하는 방식이다.

훈련-검증-테스트 구조

Training set ← Model → Validation set → Test set:

  • 여기서는 모델을 훈련 세트에서 학습한 후, 검증 세트(Validation set)로 성능을 확인하고 모델을 최적화한 뒤, 마지막으로 테스트 세트에서 성능을 평가한다. 검증 세트는 과적합을 방지하고 최적의 하이퍼파라미터를 선택하는 데 사용된다.

교차 검증(K-fold Cross-Validation)

K-fold CV:

  • 이 부분은 K-겹 교차 검증(K-fold Cross-Validation)을 보여준다. 데이터를 여러 개의 부분으로 나누고, 각 부분을 한 번씩 검증 세트로 사용하며 나머지 부분을 훈련 세트로 사용한다. 이렇게 모든 부분이 검증 세트로 사용될 때까지 이 과정을 반복한다. 이 방법은 데이터가 적을 때 모델 성능을 더 정확하게 평가할 수 있는 방법이다.
