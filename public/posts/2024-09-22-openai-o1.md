# OpenAI o1: 추론하는 AI의 등장

OpenAI가 [o1 모델](https://openai.com/o1/)을 공개했습니다. 기존 GPT 시리즈와는 설계 방향 자체가 다릅니다.

---

## GPT-4와 무엇이 다른가

GPT-4는 질문을 받으면 바로 답변을 생성합니다. o1은 답변하기 전에 내부적으로 생각하는 과정을 거칩니다.

이 추론 과정을 **Chain-of-Thought(CoT)** 라고 하는데, o1은 CoT를 강화 학습으로 직접 최적화했습니다. 모델이 더 오래 생각할수록 더 정확한 답을 낸다는 아이디어입니다.

```
GPT-4: 질문 → 바로 답변
o1:    질문 → [내부 추론 과정] → 답변
```

---

## 어떤 문제에 강한가

복잡한 수학 문제, 논리 추론, 코딩처럼 단계적 사고가 필요한 태스크에서 GPT-4o보다 눈에 띄게 높은 성능을 보입니다. 수학 올림피아드 문제나 어려운 코딩 인터뷰 문제에서 특히 차이가 납니다.

반면 단순한 질문에는 오버킬이고, 응답 속도가 느리고 비용도 높습니다. 모든 상황에 쓰기보다 복잡한 문제에 선택적으로 쓰는 게 맞습니다.

---

## Test-time Compute라는 개념

흥미로웠던 건 AI 성능을 높이는 방향이 바뀌고 있다는 점입니다.

기존에는 "더 큰 모델 = 더 좋은 성능"이었습니다. o1은 "더 많이 생각하게 하면 = 더 좋은 성능"이라는 방향을 보여줬습니다. 학습 비용을 높이는 대신 추론 시간에 컴퓨팅을 쓰는 방식입니다(Test-time Compute).

LLM 성능 향상의 방법이 모델 크기 외에 하나 더 생긴 셈입니다.
