# RentEase

## 프로젝트 개요

- **개발 기간**: 2024.07 ~ 2024.08
- **기술 스택**: React, NestJS, MySQL
- **프로젝트 유형**: 개인 프로젝트

> [Repository](https://github.com/1Dohyeon/RentEase)

## 왜 만들었는가

취미가 생길 때마다 장비를 구매하는 것은 부담이 크고, 쓰지 않을 때 보관도 문제입니다. 기존 렌탈 서비스는 대부분 업체 중심이라 개인이 보유한 장비를 유휴 시간에 빌려주거나 빌릴 수 있는 P2P 채널이 없었습니다. 이 공백을 채우는 개인 간 장비 대여 플랫폼을 개인 프로젝트로 기획했습니다.

## 핵심 설계 결정

C2C 플랫폼에서 가장 중요한 것은 두 가지입니다. 원하는 물건을 찾는 것과, 낯선 사람과 거래를 성사시키는 것입니다.

첫 번째는 **지역 기반 검색**으로 해결했습니다. 대여 특성상 멀리 있는 물건은 의미가 없기 때문에, 사용자가 프로필에서 지역을 설정하면 해당 지역의 게시글만 노출됩니다. 물건을 올릴 때는 이미지, 렌탈 가격, 대여 조건을 함께 등록합니다.

두 번째는 **WebSocket 기반 실시간 1:1 채팅**으로 해결했습니다. 게시글에서 바로 채팅을 시작할 수 있고, 대화 내역은 저장되어 거래 과정을 추적할 수 있습니다. HTTP 폴링 대신 WebSocket을 선택한 이유는 C2C 거래 특성상 빠른 응답이 거래 성사에 직접 영향을 미치기 때문입니다.

백엔드 첫 개인 프로젝트로 RESTful API 설계부터 ERD 모델링, 실시간 통신까지 전 과정을 직접 구현했습니다.
